Pour une installation propre 
1- Nouveau Dossier/Projet :
create-react-app jour5
2 – Entrer dans le Nouveau dossier
cd jour5
3- Installer Bootstrap
npm i bootstrap
4 –installer Axios
npm i axios
5 – installer le routeur
npm i react-router-dom
6- Démarrer le Projet
npm start

•    Découverte api context (un state global appelé store) https://reactjs.org/docs/context.html. On fait une classe provider dans un fichier context.js qui va nous permettre d’avoir un beau state partagé entre composants et un consumer pour utiliser ces données. Pour utiliser les données intelligemment (là ou y’en a besoin) on va créer un consumer  à importer par les classes qui ont besoin.
